## 컴파일 타임 의존관계 VS 런타임 의존관계
- 컴파일 타임 의존관계
  - 코드 컴파일 시점
  - 컴파일 시점에 어떤 클래스/인터페이스를 의존하는지
- 런타임
  - 프로그램 실행 시점
  - 실제로 어떤 구현체 인스턴스를 의존하는지
