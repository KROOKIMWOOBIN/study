#### 리스트
순서가 있고 중복을 허용하는 자료 구조를 리스트라 한다.
- [배열리스트](./library/ArrayList.md)
- [연결리스트](./library/LinkedList.md)
#### 성능 비교 표
| 기능        | 배열리스트 | 연결리스트 |
|-----------|-------|-------|
| 인덱스 조회    | O(1)  | O(n)  |
| 검색        |O(n)|O(n)|
| 앞에 추가(삭제) |O(n)|O(1)|
| 뒤에 추가(삭제) |O(1)|O(n)|
| 평균 추(삭제)  |O(n)|O(n)|
