## 큐
- 선입 천출(FIFO, First In First Out)

### Queue Method()
- offer()
  - 데이터를 넣는다.
- poll()
  - 데이터를 꺼낸다.

### Collection 
- Queue
  - Deque
    - ArrayDeque
    - LinkedList

### Deque
- Deque는 "Double Ended Queue"의 약자로, 양쪽 끝에서 요소를 추가하거나 제거할 수 있다.

#### Deque Method()
- offerFirst()
  - 앞에 추가한다.
- offerLast()
  - 뒤에 추가한다.
- pollFirst()
  - 앞에서 꺼낸다.
- pollLast()
  - 뒤에서 꺼낸다.
