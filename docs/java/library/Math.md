## Math Class
- Math 클래스는 Java Standard Library 에 포함된 수학 연산 전용 유틸리티 클래스다.
- 모든 기능이 `정적(static)`으로 제공되며, 객체 생성 없이 사용한다.

### 핵심 특징
- 정적 메서드만 존재
- 불변 클래스 (final)
- 고정된 수학 연산 제공

### 기본 연산 / 비교
| 메서드              | 설명  | 예시               | 결과   |
| ---------------- | --- | ---------------- | ---- |
| `Math.abs(x)`    | 절댓값 | `Math.abs(-10)`  | `10` |
| `Math.max(a, b)` | 최대값 | `Math.max(3, 7)` | `7`  |
| `Math.min(a, b)` | 최소값 | `Math.min(3, 7)` | `3`  |
