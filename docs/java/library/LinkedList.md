## 연결리스트
### 연결 리스트와 빅오
#### 조회
```java
Object get(int value);
```
- 특정 위치에 있는 데이터를 반환한다.
- O(n)
  - 연결 리스트에서 사용되는 노드들은 다음 노드의 대한 참조가 있을 뿐이다. 따라서 인덱스로 원하는 위치를 찾을려면 인덱스 숫자만큼 다음 노드를 반복해서 찾아야 한다.
  - 특정 위치에 노드를 찾는데 O(n)이 걸린다.
#### 추가
```java
void add(Object e);
```
- 마지막에 데이터를 추가한다.
- O(n)
  - 마지막 노드를 찾는데 O(n)이 소요된다. 마지막 노드에 새로운 노드를 추가하는데 O(1)이 걸린다. 따라서 O(n)이다.
#### 변경
```java
Object set(int index, Object element);
```
- 특정 위치에 있는 데이터를 찾아서 변경한다. 그리고 기존 값을 반환한다.
- O(n)
  - 특정 위치에 노드를 찾는데 O(n)이 걸린다.
#### 검색
```java
int indexOf(Object o);
```
- 데이터를 검색하고, 검색된 위치를 반환한다.
- O(n)
  - 모든 노드 순회하면서 equals()를 사용해서 같은 데이터가 있는지 찾는다.
