## 래퍼 클래스
- 래퍼 클래스는 기본형(Primitive Type)을 객체(Object)로 감싸는 클래스다.

### 기본형 ↔ 래퍼 클래스 매핑
| 기본형 (Primitive) | 래퍼 클래스      |
| --------------- | ----------- |
| `byte`          | `Byte`      |
| `short`         | `Short`     |
| `int`           | `Integer`   |
| `long`          | `Long`      |
| `float`         | `Float`     |
| `double`        | `Double`    |
| `char`          | `Character` |
| `boolean`       | `Boolean`   |

### 왜 필요한가?
- 자바 컬렉션은 객체만 저장 가능하다.
```java
List<Integer> list = new ArrayList<>();
list.add(10);
```
- 제네릭 타입도 참조 타입만 허용한다.
```java
class Box<T> {
    T value;
}
Box<Integer> box = new Box<>();
```
- 유틸리티 메서드 제공
```java
Integer.parseInt("10");
Integer.max(10, 20);
```

### 오토박싱 / 언박싱 지원
- Java 5부터 자동 변환 기능이 추가되었다.
- 오토박싱 (Primitive → Wrapper)
```java
Integer num = 10; // 코드
-> Integer num = Integer.valueOf(10);
```
- 언박싱 (Wrapper → Primitive)
```java
Integer num = 10;
-> int value = num;
```
