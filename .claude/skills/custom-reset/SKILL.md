이 스킬이 호출되면 아래 형식으로 보고하세요.

---

## 세션 요약

현재 대화에서 수행한 작업을 간략히 정리합니다:
- 완료한 작업 목록
- 주요 결정 사항
- 미완료 작업 (있다면)

---

## 컨텍스트 상태

- 자동 압축(Auto-compact) 발생 여부
- 대화 밀도 추정 (가볍다 / 보통 / 무겁다)

---

## 권장 사항

컨텍스트 사용량에 따라 아래 중 하나를 권장합니다:
- 아직 여유 있음 → 계속 진행 가능
- 중간 수준 → 조만간 `/compact` 권장
- 포화 상태 → 지금 `/compact` 실행 권장

---

## 남은 토큰 추정

Claude는 자신의 정확한 토큰 수를 알 수 없으므로, **컨텍스트 윈도우 기준으로 추정**합니다.

- 현재 모델 컨텍스트 윈도우: 200,000 tokens (claude-sonnet-4-6 / claude-opus-4-6 기준)
- 사용된 컨텍스트 비율: [낮음 / 중간 / 높음] (대화 길이와 첨부 파일 기준으로 주관적 추정)
- 추정 잔여 토큰: [여유 있음 / 주의 필요 / 즉시 compact 권장]

> 정확한 토큰 수는 `/cost` 명령으로 누적 비용을 확인하거나, `/compact` 실행 전 Claude Code가 표시하는 컨텍스트 사용량을 참고하세요.
