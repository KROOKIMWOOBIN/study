## 학습 노트 정리 필수 항목

`docs/` 하위 파일에 개념을 정리할 때 아래 항목을 반드시 포함한다.

| 항목 | 설명 |
|------|------|
| 왜 쓰는지 | 이 개념/기능이 존재하는 이유, 해결하는 문제 |
| 언제 쓰는지 | 적합한 사용 상황, 조건 |
| 장점 | 사용했을 때의 이점 |
| 단점 | 한계, 트레이드오프 |
| 특징 | 다른 것과 구별되는 핵심 성질 |
| 주의할 점 | 실수하기 쉬운 부분, 안티패턴 |

모든 항목이 항상 별도 섹션일 필요는 없다. 테이블, 인라인 설명 등 형식은 자유롭게 사용하되 내용은 빠짐없이 담는다.

## 주제가 기존 파일과 겹칠 때

새 주제가 기존 파일에서 이미 다루고 있는 내용과 겹치더라도, **별도 파일로 더 깊게 작성**한다. 링크로 대체하거나 항목을 제거하지 않는다.

- 기존 파일: 개요 수준으로 언급
- 새 파일: 해당 주제만 집중해서 심화 정리
