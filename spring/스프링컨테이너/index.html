    мҠӨн”„л§Ғ м»Ён…Ңмқҙл„ҲлһҖ?

    мҠӨн”„л§Ғ м»Ён…Ңмқҙл„ҲлҠ” л№Ҳ(Bean)мқҳ мғқм„ұВ·кҙҖлҰ¬В·мқҳмЎҙкҙҖкі„ мЈјмһ…мқ„ лӢҙлӢ№н•ҳлҠ” н•өмӢ¬ кҙҖлҰ¬мһҗлӢӨ.

    к°ңл°ңмһҗк°Җ AppConfigм—җ м„Өм • м •ліҙлҘј мһ‘м„ұн•ҳл©ҙ, мҠӨн”„л§Ғ м»Ён…Ңмқҙл„Ҳк°Җ мқҙлҘј мқҪм–ҙм„ң л№Ҳмқ„ л“ұлЎқн•ҳкі  DIлҘј мІҳлҰ¬н•ңлӢӨ.

    // мҠӨн”„л§Ғ м»Ён…Ңмқҙл„Ҳ мғқм„ұ вҖ” AppConfig м„Өм • м •ліҙлҘј мқҪм–ҙм„ң л№Ҳмқ„ л“ұлЎқ
ApplicationContext ac = new AnnotationConfigApplicationContext(AppConfig.class);

// л“ұлЎқлҗң л№Ҳ кәјлӮҙ м“°кё°
OrderService orderService = ac.getBean("orderService", OrderService.class);

    ApplicationContextк°Җ мҠӨн”„л§Ғ м»Ён…Ңмқҙл„Ҳмқҳ мқён„°нҺҳмқҙмҠӨлӢӨ.

    BeanFactory vs ApplicationContext

    BeanFactory

    мҠӨн”„л§Ғ м»Ён…Ңмқҙл„Ҳмқҳ мөңмғҒмң„ мқён„°нҺҳмқҙмҠӨ. л№Ҳмқ„ кҙҖлҰ¬н•ҳкі  мЎ°нҡҢн•ҳлҠ” кё°ліё кё°лҠҘмқ„ м ңкіөн•ңлӢӨ.

    • л№Ҳмқ„ м§Җм—° лЎңл”©(Lazy Loading): getBean() нҳём¶ң мӢңм җм—җ мғқм„ұ
    • мӢӨл¬ҙм—җм„ң м§Ғм ‘ м“ё мқјмқҖ кұ°мқҳ м—ҶлӢӨ

    ApplicationContext

    BeanFactoryлҘј мғҒмҶҚн•ҳкі  мӢӨл¬ҙм—җ н•„мҡ”н•ң л¶Җк°Җ кё°лҠҘмқ„ м¶”к°Җн•ң мқён„°нҺҳмқҙмҠӨ. н•ӯмғҒ мқҙкІғмқ„ мӮ¬мҡ©н•ңлӢӨ.

    ApplicationContext мғҒмҶҚ кі„мёө:
в”ңв”Җв”Җ BeanFactory              (л№Ҳ мЎ°нҡҢ, кҙҖлҰ¬)
в”ңв”Җв”Җ EnvironmentCapable       (нҷҳкІҪліҖмҲҳ вҖ” local/dev/prod кө¬л¶„)
в”ңв”Җв”Җ MessageSource            (көӯм ңнҷ” вҖ” messages.properties)
в”ңв”Җв”Җ ApplicationEventPublisher (мқҙлІӨнҠё л°ңн–ү/кө¬лҸ…)
в””в”Җв”Җ ResourceLoader           (нҢҢмқј/нҒҙлһҳмҠӨнҢЁмҠӨ/URL лҰ¬мҶҢмҠӨ лЎңл”©)
    кё°лҠҘ BeanFactory ApplicationContext
    л№Ҳ кҙҖлҰ¬ O O
    нҷҳкІҪліҖмҲҳ (н”„лЎңнҢҢмқј) X O
    көӯм ңнҷ” X O
    мқҙлІӨнҠё X O
    лҰ¬мҶҢмҠӨ лЎңл”© X O
    л№Ҳ лЎңл”© л°©мӢқ м§Җм—°(Lazy) мҰүмӢң(Eager) вҖ” кё°ліё

    мҠӨн”„л§Ғ м»Ён…Ңмқҙл„Ҳ мғқм„ұ кіјм •

    1лӢЁкі„: м»Ён…Ңмқҙл„Ҳ мғқм„ұ

    ApplicationContext ac = new AnnotationConfigApplicationContext(AppConfig.class);

    [мҠӨн”„л§Ғ м»Ён…Ңмқҙл„Ҳ]

    л№Ҳ мқҙлҰ„  вҶ’  л№Ҳ к°қмІҙ
    в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
    (м—ҶмқҢ)

    2лӢЁкі„: л№Ҳ л“ұлЎқ

    AppConfigлҘј мқҪм–ҙм„ң @Beanмқҙ л¶ҷмқҖ л©”м„ңл“ңлҘј лӘЁл‘җ нҳём¶ңн•ҳкі , л°ҳнҷҳлҗң к°қмІҙлҘј л№ҲмңјлЎң л“ұлЎқн•ңлӢӨ.

    @Configuration
public class AppConfig {
    @Bean
    public MemberService memberService() { return new MemberServiceImpl(memberRepository()); }

    @Bean
    public MemberRepository memberRepository() { return new MemoryMemberRepository(); }

    @Bean
    public OrderService orderService() { return new OrderServiceImpl(memberRepository(), discountPolicy()); }

    @Bean
    public DiscountPolicy discountPolicy() { return new RateDiscountPolicy(); }
}

    [мҠӨн”„л§Ғ м»Ён…Ңмқҙл„Ҳ]
    л№Ҳ мқҙлҰ„           вҶ’  л№Ҳ к°қмІҙ
    в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
    memberService     вҶ’  MemberServiceImpl@x01
    memberRepository  вҶ’  MemoryMemberRepository@x02
    orderService      вҶ’  OrderServiceImpl@x03
    discountPolicy    вҶ’  RateDiscountPolicy@x04

    л№Ҳ мқҙлҰ„мқҖ кё°ліём ҒмңјлЎң л©”м„ңл“ң мқҙлҰ„мқҙлӢӨ.

    @Bean(name = "myOrderService") // мқҙлҰ„ м§Ғм ‘ м§Җм •лҸ„ к°ҖлҠҘ
public OrderService orderService() { ... }

    мЈјмқҳ: л№Ҳ мқҙлҰ„мқҙ кІ№м№ҳл©ҙ мҳҲмҷёк°Җ л°ңмғқн•ҳкұ°лӮҳ мҳӨлІ„лқјмқҙл”©лҗңлӢӨ. н•ӯмғҒ лӢӨлҘё мқҙлҰ„мқ„ мҚЁм•ј н•ңлӢӨ.

    3лӢЁкі„: мқҳмЎҙкҙҖкі„ мЈјмһ…

    м„Өм • м •ліҙлҘј м°ёкі н•ҳм—¬ л№Ҳл“Өмқҳ мқҳмЎҙкҙҖкі„лҘј м—°кІ°н•ңлӢӨ.

    [л№Ҳ к°қмІҙ м—°кІ° кІ°кіј]
MemberServiceImpl@x01
    в””в”Җв”Җ memberRepository вҶ’ MemoryMemberRepository@x02

OrderServiceImpl@x03
    в”ңв”Җв”Җ memberRepository вҶ’ MemoryMemberRepository@x02
    в””в”Җв”Җ discountPolicy   вҶ’ RateDiscountPolicy@x04

    мғқм„ұмһҗ мЈјмһ…мқҳ кІҪмҡ° л№Ҳ мғқм„ұкіј DIк°Җ лҸҷмӢңм—җ мқјм–ҙлӮңлӢӨ.

    л№Ҳ м„Өм • л°©лІ• 3к°Җм§Җ

    Java Config л°©мӢқ (нҳ„мһ¬ мЈјлҘҳ)

    @Configuration
public class AppConfig {
    @Bean
    public MemberRepository memberRepository() {
        return new MemoryMemberRepository();
    }
    // ...
}

ApplicationContext ac = new AnnotationConfigApplicationContext(AppConfig.class);

    Component Scan л°©мӢқ (Spring Boot кё°ліё)

    @SpringBootApplication // @ComponentScanмқҙ нҸ¬н•Ёлҗҳм–ҙ мһҲмқҢ
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}

    @Component, @Service, @Repository, @Controllerк°Җ л¶ҷмқҖ нҒҙлһҳмҠӨлҘј мһҗлҸҷмңјлЎң л№Ҳ л“ұлЎқ.

    XML л°©мӢқ (л Ҳкұ°мӢң)

    <!-- applicationContext.xml -->
<beans>
    <bean id="memberRepository" class="hello.core.member.MemoryMemberRepository"/>
    <bean id="orderService" class="hello.core.order.OrderServiceImpl">
        <constructor-arg name="memberRepository" ref="memberRepository"/>
    </bean>
</beans>

    мҳӨлһҳлҗң н”„лЎңм қнҠём—җм„ң ліј мҲҳ мһҲм§Җл§Ң, мӢ к·ң н”„лЎңм қнҠём—җм„  кұ°мқҳ м“°м§Җ м•ҠлҠ”лӢӨ.

    л№Ҳ мЎ°нҡҢ

    кё°ліё мЎ°нҡҢ

    ApplicationContext ac = new AnnotationConfigApplicationContext(AppConfig.class);

// мқҙлҰ„ + нғҖмһ…мңјлЎң мЎ°нҡҢ (к°ҖмһҘ лӘ…нҷ•)
MemberService memberService = ac.getBean("memberService", MemberService.class);

// нғҖмһ…л§ҢмңјлЎң мЎ°нҡҢ (к°ҷмқҖ нғҖмһ…мқҙ 2к°ң мқҙмғҒмқҙл©ҙ мҳҲмҷё л°ңмғқ!)
MemberService memberService = ac.getBean(MemberService.class);

    к°ҷмқҖ нғҖмһ… л№Ҳмқҙ м—¬лҹ¬ к°ңмқё кІҪмҡ°

    // MemberRepository кө¬нҳ„мІҙк°Җ л‘җ к°ң л“ұлЎқлҗҳм–ҙ мһҲлӢӨл©ҙ
ac.getBean(MemberRepository.class);
// вҶ’ NoUniqueBeanDefinitionException: expected single matching bean but found 2

// н•ҙкІ° 1: мқҙлҰ„ м§Җм •
ac.getBean("memoryMemberRepository", MemberRepository.class);

// н•ҙкІ° 2: н•ҙлӢ№ нғҖмһ…мқҳ лӘЁл“  л№Ҳ мЎ°нҡҢ
Map<String, MemberRepository> beansOfType = ac.getBeansOfType(MemberRepository.class);
beansOfType.forEach((name, bean) -> System.out.println("name=" + name + " bean=" + bean));

    мғҒмҶҚ кҙҖкі„ мЎ°нҡҢ мЈјмқҳ

    л¶ҖлӘЁ нғҖмһ…мңјлЎң мЎ°нҡҢн•ҳл©ҙ мһҗмӢқ нғҖмһ… л№ҲлҸ„ лӘЁл‘җ мЎ°нҡҢлҗңлӢӨ.

    // DiscountPolicyлҘј кө¬нҳ„н•ң л№Ҳмқҙ Fix, Rate л‘җ к°ң мһҲмқ„ л•Ң
ac.getBean(DiscountPolicy.class);
// вҶ’ NoUniqueBeanDefinitionException!

// мқҙлҰ„ м§Җм • н•„мҡ”
ac.getBean("rateDiscountPolicy", DiscountPolicy.class);

    ObjectлЎң мЎ°нҡҢн•ҳл©ҙ мҠӨн”„л§Ғмқҙ л“ұлЎқн•ң лӘЁл“  л№Ҳмқҙ лӮҳмҳЁлӢӨ. (лӮҙл¶Җ мҠӨн”„л§Ғ л№Ҳ нҸ¬н•Ё)

    л№Ҳ л“ұлЎқ л°©лІ• л№„көҗ

    л°©лІ• нҠ№м§• мӮ¬мҡ© мӢңкё°
    @Bean (Java Config) лӘ…мӢңм Ғ л“ұлЎқ мҷёл¶Җ лқјмқҙлёҢлҹ¬лҰ¬, кіөнҶө кё°мҲ  л№Ҳ
    @Component + @ComponentScan мһҗлҸҷ л“ұлЎқ мқјл°ҳ л№„мҰҲлӢҲмҠӨ л№Ҳ
    XML л Ҳкұ°мӢң мҳӨлһҳлҗң н”„лЎңм қнҠё мң м§ҖліҙмҲҳ
